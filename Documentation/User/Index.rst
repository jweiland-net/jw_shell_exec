============
Users Manual
============

Execute Shell Script
====================

..  rst-class:: bignums

1.  Visit BE Module

    Click on `JW Shell Exec` from the left main menu in TYPO3 backend. It is only visible, if an integrator
    has given you access to that BE module.

2.  Execute

    If you are the only user logged in into the TYPO3 BE you can now execute the defined shell script.
