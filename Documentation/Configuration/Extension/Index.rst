==================
Extension Settings
==================

Some general settings for `jw_shell_exec` can be configured in *Admin Tools -> Settings*.


Tab: Basic
==========

Shell Script
------------

Default: <empty>

Define an absolute path to an executable Shell Script

It is also allowed to start the script path with `EXT:`.

Do not append any aguments to the script path!
